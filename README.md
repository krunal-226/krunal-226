<a name="projects"></a>

## 🚀 Projects

### 🛡️ THE BLADE — Network Security Auditing Tool
A Python-based network security auditing tool built with my team at a Hackathon.

**My contribution — Multi-Scan Management System:**
- ✅ Auto-incrementing scan folders (`scan_1`, `scan_2`, `scan_3`...) so results from different scans never overwrite each other
- ✅ Organized result storage with structured JSON reports per scan — `open_ports.json`, `closed_ports.json`, `filtered_ports.json`, `result.json`, `scan_info.json` (IP + timestamp)
- ✅ Scan history tracking, logging which IP was scanned and when

This makes THE BLADE capable of managing multiple audits across different targets without losing previous scan data — built for real enterprise-style network assessments.

Built with teammates: Joy Sanjaykumar Patel, Om Rathwa, Nirmalsharan Patel, and Noel Messey.

➜ [GitHub Repo](https://github.com/joypatel201007/Socket-Scan.git)

### 🎓 Student Productivity Hub
An all-in-one productivity dashboard built specifically for students, so they don't have to switch between separate apps for notes, timers, and tasks and lose focus.

**Key Features:**
- ⏱️ **Pomodoro Timer** — for deep focus using time-blocking
- ✅ **Dynamic To-Do List** — to track assignments and daily goals
- 📝 **Quick Notes** — a dedicated space for capturing thoughts on the fly

➜ [GitHub Repo](https://github.com/krunal-226/Student-productivity-hub.git)
